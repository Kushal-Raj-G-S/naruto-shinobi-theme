## Installation

### Marketplace

1. Open VS Code
2. Go to Extensions (`Ctrl+Shift+X`)
3. Search `Naruto Shinobi Theme`
4. Install
5. Run `Preferences: Color Theme`
6. Choose any Naruto variant

### VSIX

```bash
code --install-extension naruto-shinobi-theme-1.1.0.vsix
```

## Recommended Settings

Add this to your user settings JSON:

```jsonc
{
  "workbench.colorTheme": "Naruto Shinobi Night",
  "editor.fontFamily": "'Fira Code', 'Cascadia Code', 'JetBrains Mono', Consolas, monospace",
  "editor.fontLigatures": true,
  "editor.fontSize": 14,
  "editor.lineHeight": 22,
  "editor.cursorBlinking": "smooth",
  "editor.cursorSmoothCaretAnimation": "on",
  "editor.bracketPairColorization.enabled": true,
  "editor.semanticHighlighting.enabled": true,
  "terminal.integrated.fontFamily": "'Cascadia Code', 'Fira Code', Consolas"
}
```

## Quick Customization

```jsonc
{
  "workbench.colorCustomizations": {
    "[Naruto Shinobi Night]": {
      "statusBar.background": "#FF6B00"
    }
  },
  "editor.tokenColorCustomizations": {
    "[Naruto Akatsuki]": {
      "comments": "#8A6666"
    }
  }
}
```
