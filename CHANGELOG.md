# Changelog

All notable changes to the "Naruto Shinobi Theme" extension will be documented in this file.

## [1.0.0] - 2025-12-08

### Added
- 🎉 Initial release of Naruto Shinobi Theme
- 🌙 **Naruto Shinobi Night** - High contrast dark theme with vibrant orange and blue accents
- 🌅 **Naruto Hokage Dawn** - Softer dark theme with balanced contrast and warm tones
- 🎨 Complete color palette inspired by Naruto universe
- 💎 Semantic highlighting support for modern languages
- 🔧 Comprehensive syntax highlighting for:
  - JavaScript/TypeScript
  - React/JSX/TSX
  - Python
  - HTML/CSS
  - JSON
  - Markdown
  - And many more languages
- 📚 Full UI theming (Activity Bar, Status Bar, Sidebar, Panels, etc.)
- 🎯 Optimized terminal colors
- 🔍 Enhanced Git decoration colors
- 📖 Detailed documentation and color reference

### Features
- Uzumaki Orange (#FF6B00) as primary accent
- Chakra Blue (#4A90E2) for secondary highlights
- Sharingan Red (#CC0000) for errors
- Nine-Tails Gold (#FFD700) for constants
- Rinnegan Purple (#9C27B0) for functions
- Forest Green (#4CAF50) for strings
- Rasengan Cyan (#00CED1) for classes
